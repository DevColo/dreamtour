# Dream Destinations Tours & Travel

Premium African Safari experiences across Uganda, Rwanda, Kenya, and Tanzania.

## About

Dream Destinations Tours & Travel specializes in crafting bespoke safari journeys, from gorilla trekking to the Great Migration. We create unforgettable experiences in the heart of Africa.

## Technologies

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Development

```sh
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
